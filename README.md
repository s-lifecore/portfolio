## 🌟 Identity: 防災 × IT

私は、専門である情報工学と、情熱を注いでいる防災活動を融合させた「シビックテック」の社会実装に注力しています。

### 🛰️ リアルタイム防災バナーシステム (New!)

ビジネスカードページ（`/hello/business-card/*`）に搭載された、訪問者の安全を守るための動的機能です。

- **パーソナライズ**: 訪問者のIPアドレスから地域を特定し、リアルタイムな気象警報・注意報を表示。
- **イベント連携**: Supabaseを活用し、ハッカソンや登壇などのイベントスケジュールに合わせた動的なメッセージ切り替え。
- **信頼性の担保**: GitHub Actionsによる定期PingとMarkdownログ記録（DMD構想）による死活監視。

## 🛠️ Featured Projects

### 1. CityRiskView

**Disaster Resilience Platform**
災害時の迅速な情報共有と、平時の備えを支援する防災プラットフォーム。リアルタイムな脆弱性データの可視化を目指しています。

### 2. KIT DevelopersHub & GDGoC KIT

**Student Hackathon Organizer**
学生向けハッカソンの企画・運営や、学内外の技術コミュニティ活動を通じて、若手エンジニアの育成と官民連携を推進しています。

### 3. Michikusa Memory

**Walking Memory Archive**
「ふとした散歩を冒険に変える」をコンセプトに、日常の移動を記録し価値化するアプリケーション。
